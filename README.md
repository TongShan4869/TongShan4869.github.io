# TongShan4869.github.io

Personal website — built with Next.js, Tailwind CSS, Framer Motion, and Keystatic CMS.

## Development

```bash
cd next
npm install
npm run dev
```

Visit `http://localhost:3000` for the site and `http://localhost:3000/keystatic` for the CMS admin.

## Build

```bash
cd next
npm run build
```

Static output is generated in `next/out/`.

## Deployment

Deployed to GitHub Pages via GitHub Actions on push to `rebuild/nextjs` branch.
